# Phase Vocoder

Phase vocoder implementation in Python 3 for time-stretching and pitch-shifting.

## Dependencies 

- Python 3 (I used Python 3.11.3)
- Numpy (run `pip install numpy`)
- Scipy (run `pip install scipy`)
